# Contributing

This repository is maintained with a maintainer-led workflow for Open Almond Studios.

## External Contributions

Public pull requests are not accepted by default.

If you want to suggest a change, open an issue first with:

- The problem statement
- The proposed change
- Any CMS or production risks

We may invite targeted contributions on a case-by-case basis.

## Internal Workflow

For maintainers, use small phased changes and verify each phase:

1. `npm run build`
2. Local smoke test on core routes (`/`, `/blog`, key content pages)
3. Confirm no Notion rendering regressions
4. Confirm no new console/runtime errors
